## 介绍
**PyPI**上似乎还没有功能完整的魔方模拟器包，导致我们在写关于魔方的程序的时候发现少了一些功能，所以我就做了一个包含以下功能的Python Package:
 1. 公式：化简，反转以及其它公式操作。
 2. 魔方：模拟，求解以及其它魔方操作。

## 包概述
### 模块
1. `cube.py`: 模拟魔方。
2. `solver.py`: 求解魔方。
3. `alg.py`: 操作公式。
